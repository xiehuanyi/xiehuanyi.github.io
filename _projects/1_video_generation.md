**Principal Investigator** (Aug 2023 - Sep 2023)

Developed a complete pipeline for automatic video generation from dialogue.
- **Data Retrieval**: Used web scraping techniques to retrieve background information data on specified topics for Retrieval Augmented Generation.
- **Content Generation**: Designed stable and personalized prompts for large language models using prompt engineering techniques.
- **System Pipeline**: Designed a pipeline for script generation, video generation, and TTS (Text-to-Speech).
- **Interface**: Built a frontend user interface using Gradio.
